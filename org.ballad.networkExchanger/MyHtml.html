Network Exchanger 帮助文档

V1.0 Beta

Network Exchanger是一款可以方便管理Windows的IP网络配置方案的软件。

运行环境要求：
Windows XP/Vista/7
JRE 1.5或更高版本

Network Exchanger基于Java™平台开发，所以请确保您的计算机上已经安装了Java™运行环境。
获取Java™运行环境（JRE）： http://www.java.com/download/

感谢如下技术及平台：

  • Java SE
  • Eclipse IDE
  • NetBeans IDE
  • App Framework
  • Swing Works
